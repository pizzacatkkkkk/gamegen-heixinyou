# 測試紀錄 — 2026-07-23 語音＋打擊感＋連發改版

## 這次改了什麼
1. **台詞語音**：14 段 edge-tts 港腔國語（zh-HK-WanLungNeural，rate +12%）base64 內嵌
   - 叫囂 t0-t5（對應 villainTaunts）、慘叫 p0-p5、開場 start、告饒 win
   - 內建節流：上一句沒講完不搶話；告饒 force 蓋台
2. **台詞氣泡**：固定畫面中間（W/2, H*0.36），加大加邊框，尾巴自動指向商人
3. **連擊大字**：移到畫面下方（H*0.68），命中彈跳、20 連以上變紅
4. **打擊感**：漫畫星爆「啪／碰／咚」、畫面縮放脈衝、油漬噴濺殘留、拋射物殘影、
   連擊里程碑（每 10 連大震＋「太狠了！」）、手機震動、震幅隨連擊成長
5. **按住連發**：按住 0.25s 進入連發（0.14s/發），可拖移瞄準、附脈動準星；快速點放仍是單發
6. **修正**：miss 不再歸零連擊（連發流彈誤傷手感）；hit-stop 節流 220ms（連發不卡頓）；
   villain.maxScale 每幀重算（開局版面未就緒時自我修復）；移除無效 CSS `active: none`；
   新增靜音鈕（localStorage 記憶）；輸入改 pointer events（原 mouse+touch 雙軌）
